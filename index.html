Meus Projetos

Barbearia Chico Leme

Sistema completo de agendamento de clientes para uma barbearia real. Desenvolvido como TCC. Integra funcionalidades como cadastro de horários, dados do cliente e painel de administração.

Bots para Discord

Um Bot no Discord, fundamentalmente, é um programa automatizado projetado para realizar diversas tarefas que enriquecem a experiência em servidores Discord. Esses bots atuam como assistentes virtuais, trazendo uma gama de funcionalidades que simplificam e dinamizam a administração e a interação dentro dos servidores.

PDV Parcial

Sistema para Ponto de Venda, de empresas de pequeno à médio porte. PDV é o local onde as transações comerciais ocorrem, seja em uma loja física, um site ou outro canal de vendas.

Fish Delivery

Um aplicativo de delivery de comida desenvolvido para praticar conceitos de front-end e back-end. O sistema permite que usuários naveguem por um cardápio digital interativo, escolham pratos, adicionem ao carrinho e finalizem o pedido.

Login

Sistema de login com autenticação visual. Ideal para testes de formulário, validação de campos e responsividade. Pode conter tela de cadastro e recuperação de senha.
