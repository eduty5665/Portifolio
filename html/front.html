Projetos Front-End

Academia

Um site institucional para uma academia de musculação. Apresenta seções sobre planos, serviços, horários, galeria e contato. Design moderno e focado em promover saúde e bem-estar.

Weather App

Aplicação web para consultar a previsão do tempo em tempo real. Interface limpa, entrada de cidade/localização e exibição de dados como temperatura, umidade e clima usando API externa.

Site do Mario

Projeto visual e divertido inspirado no universo do Super Mario. Pode incluir animações, personagens e ambientação de jogos clássicos da franquia Nintendo.

Curriculo Web

Portfólio pessoal baseado no modelo Origamid. Apresenta seus projetos, tecnologias dominadas, experiência e formas de contato. Clean e profissional.

PokeDev

Projeto com temática Pokémon, talvez uma Pokédex interativa ou catálogo de personagens usando APIs. Voltado a praticar chamadas assíncronas e manipulação de dados com JavaScript.

Login

Sistema de login com autenticação visual. Ideal para testes de formulário, validação de campos e responsividade. Pode conter tela de cadastro e recuperação de senha.

Barbearia

Página institucional de uma barbearia tradicional. Contém informações sobre serviços (barba, cabelo, bigode), equipe e agendamento. Estilo clássico e masculino.

Barbearia Chico Leme

Sistema completo de agendamento de clientes para uma barbearia real. Desenvolvido como TCC. Integra funcionalidades como cadastro de horários, dados do cliente e painel de administração.

Pizzaria Romana

Site fictício para uma pizzaria italiana. Possui cardápio interativo, fotos das pizzas, história da pizzaria e formulário para pedidos/reservas.

Restaurante

Página elegante para um restaurante de culinária moderna. Destaque para menus do dia, pratos especiais, horários de funcionamento e reservas.

Cafeteria

Projeto de uma cafeteria artesanal. Interface leve e aconchegante com cardápio de cafés, lanches e sobremesas. Foco na estética e experiência do cliente.

Company

Site corporativo para empresa fictícia de tecnologia ou consultoria. Estrutura clássica com seções de “Sobre nós”, “Serviços”, “Equipe” e “Contato”.
